# Transicion-Demogrofica-y-Economia-Dashboard-de-Nashly-Marcanaupa
Este dashboard presenta un análisis interactivo sobre la relación entre la transición demográfica y el desarrollo económico mundial utilizando la base de datos Gapminder. El objetivo es explorar cómo variables demográficas y económicas evolucionan en distintos países y continentes a lo largo del tiempo, permitiendo identificar patrones de crecimiento, desigualdad y desarrollo.
El panel incluye indicadores principales como población total, esperanza de vida y PIB per cápita, los cuales permiten observar la situación económica y demográfica del país seleccionado. Además, incorpora un mapa mundial interactivo que muestra la distribución geográfica de los países según su nivel económico y tamaño poblacional, facilitando la comparación entre regiones del mundo.
El gráfico de dispersión analiza la relación entre variables demográficas y económicas, como la esperanza de vida y el PIB per cápita, permitiendo identificar correlaciones y diferencias estructurales entre continentes. El tamaño de las burbujas representa la población de cada país, mientras que los colores distinguen las regiones geográficas.
La sección de evolución histórica muestra la trayectoria temporal de las principales variables económicas del país seleccionado, permitiendo observar tendencias de crecimiento, cambios demográficos y procesos de desarrollo económico a través de los años. Complementariamente, el gráfico de barras presenta el ranking de los países con mayores niveles económicos dentro de cada continente seleccionado.
El dashboard también incorpora un análisis de regresión lineal para evaluar la relación estadística entre esperanza de vida y PIB per cápita, mostrando cómo mejoras en indicadores sociales suelen estar asociadas con mayores niveles de ingreso. Asimismo, se incluyen proyecciones económicas basadas en tendencias históricas, las cuales permiten estimar el posible comportamiento futuro del PIB per cápita.
Finalmente, la matriz de correlación resume la intensidad y dirección de la relación entre las principales variables del estudio, como población, esperanza de vida y producción económica, proporcionando una visión integral del vínculo entre dinámica demográfica y desempeño económico global.
